Introduction

Sortable DL is a jQuery plugin to turn a definition list into a sortable table. Sortable DL was born out of the need for a sortable table with the ability to open a toggle layer under a corresponding table row. Choose this plugin if you need something similar. If you just need a sortable table functionality, check out the awesome solutions out there.

Download

Sortable DL - plugin, documentation, demo: sortable-dl.zip.

Documentation

The JavaScript

To use Sortabel DL include jQuery and the Sortable DL plugin inside the <head> area of your HTML document.

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>
<script src="sortable-dl.min.js"></script>

Insert a configuration script in the <head> area of your HTML document.

<script>
var show = function(num) {
  $('#sortable dl').each(function() {
	var dd = $(this).children('dd');
	var tr = $(this).parents('tr');
	if ( (tr.attr('id') === 'row' + num) && $(this).children('dd').css('display') === 'none' ) {
      $(this).children('dd').removeClass('hide').addClass('ib');
    } else {
      $(this).children('dd').removeClass('ib').addClass('hide');
    }
  });
};

$(document).ready(function() {  

  var mySorter = new SortableDL();
  mySorter.init({
    id: 'sortable',       // Table ID
    first: 'ascending',   // optional / default 'descending'
    columns: [
      null,               // Col 1
      'caseInsensitive',  // Col 2
      'caseSensitive',    // Col 3
      'caseInsensitive',  // Col 4
      'numeric',          // Col 5
      'date',             // Col 6  
      null                // Col 7
    ],
    classes: {
      asc: 'asc',         // optional / default: 'asc'
      desc: 'desc',       // optional / default: 'desc'
      column: 'active-column' // optional / default: ''
    },
    pagination: {
	  items: 5,           // Items per page
	  numbers: 5          // Visible page numbers
    }
  });  

});
</script>

Parameters

Sorting

null
Don't sort this column
caseInsensitive
Sort case insensitively
caseSensitive
Sort case sensitively
numeric
Sort by numbers
date
Sort by dates. Sort by date accepts following date formats:
  • 1298851200000 (UNIX Timestamp)
  • 2011/02/28 (US Short Date Format)
  • 28/02/2011 (EU Short Date Format)
  • 2011-02-28T00:00:00-00:00 (ISO 8601 Date Format)
  • Mon Feb 28 2011 00:00:00 GMT+0000 (Javascript Default Date Format)
  • Mon, 28 Feb 2011 00:00:00 GMT (Javascript GMT Date Format)

CSS classes

asc
set 'asc' class on table header of sorted column when sorted ascending
desc
set 'desc' class on table header of sorted column when sorted descending

Pagination

items: n
set n items to display per page
numbers: n
n visible page numbers

The Markup

Use a definition list within a table to make sortable DL work.

Instead of:

      	<tr>
      	  <td class="col1"></td>
      	  <td class="col2"></td>
      	  <td class="col3"></td>
      	</tr>

sortable-dl takes definition list elements as columns:

      	<tr>
      	  <td>
      	    <dl>
      	      <dt class="col1"></dt>
      	      <dt class="col2"></dt>
      	      <dt class="col3"></dt>
      	    </dl>
      	  </td>
      	</tr>

To achieve a toggling info container add DD element(s)(hidden per default) :

      	<tr>
      	  <td>
      	    <dl>
      	      <dt class="col1"></dt>
      	      <dt class="col2"></dt>
      	      <dt class="col3"></dt>
      	      <dd>additional content</dd>
      	      <dd>more additional content</dd>
      	    </dl>
      	  </td>
      	</tr>

    <span class="sdl-last">Last</span>

</div> <!-- /sdl -->
